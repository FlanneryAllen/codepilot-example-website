# CodePilot Example Website

A complete marketing website example for CodePilot, an AI-powered code review platform.

## Overview

This is a fully functional example website demonstrating modern marketing site architecture with:
- Main landing page with features, pricing, testimonials
- Blog with 11 complete articles
- Responsive design
- Clean, professional UI

## Purpose

This example website was created to demonstrate narrative structure and can be used to:
- Test narrative agent analysis
- Extract messaging patterns
- Identify value propositions and positioning
- Analyze content structure and flow

## Viewing the Website

### Local Development
Simply open `index.html` in your web browser:

```bash

open index.html

# Option 2: Use file:// URL
open file:///Users/julieallen/Desktop/codepilot-example-website/index.html

```
